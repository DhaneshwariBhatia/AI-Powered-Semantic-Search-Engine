# AI Semantic Search Engine 🔍🤖

An AI-powered Semantic Search application that allows users to upload PDFs and search their content using Natural Language Queries.  
The system uses **Hugging Face embeddings + FAISS vector database + Gradio UI** to retrieve the most relevant document chunks.

---

## 🚀 Features

- 📄 Upload PDF documents
- 🧠 Convert text into embeddings using Hugging Face model
- ⚡ Store embeddings in FAISS vector database
- 🔎 Perform semantic search using natural language queries
- 📊 Retrieve top matching document chunks
- 🎨 Interactive and colorful Gradio UI

---

## 🧠 Tech Stack

- Python 🐍
- SentenceTransformers (Hugging Face)
- FAISS (Vector Database)
- PyPDF2 (PDF Processing)
- LangChain Text Splitters
- Gradio (Frontend UI)
